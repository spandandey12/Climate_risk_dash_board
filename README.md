# Global Climate Risk Intelligence Dashboard

## Project Overview

This project aims to build an interactive dashboard using Dash to analyze and compare global climate risk across countries. The dashboard will use the Climate Risk Index dataset to explore relationships between risk, fatalities, and economic losses.

---

## Objective

The main objective of this project is to design a dashboard that helps users:

- Compare climate risk between countries
- Understand the human and economic impacts of climate-related events
- Identify countries or regions with higher climate risk
=======
To design a data-driven dashboard that enables users to:

- Compare climate risk across countries  
- Analyze economic and human impacts  
- Identify high-risk regions using normalized metrics  

---

## Data Sources

- Climate Risk Index Dataset
- Resource links provided with the dataset

## Project Workflow

### Step 1: Data Understanding

- Inspect the dataset and understand the columns
- Identify important variables such as CRI score, fatalities, and economic losses

### Step 2: Data Cleaning

- Clean and standardize column names
- Check for missing values
- Convert columns to the correct data types
- Clean country names for mapping

### Step 3: Feature Engineering

- Create normalized risk measures
- Add risk categories such as low, medium, and high
- Transform economic loss values for better visualization
- Add region information if needed

### Step 4: Dashboard Design

- Create a global map to show climate risk by country
- Add country comparison charts
- Create a risk versus impact visualization
- Add a ranking table for high-risk countries

### Step 5: Dashboard Development

- Build the dashboard using Dash
- Add interactive filters such as country, region, and risk level
- Organize the layout so it is clear and easy to use

### Step 6: Insights and Interpretation

- Identify countries with higher climate risk
- Compare economic losses and fatalities
- Summarize key patterns found from the dashboard

### Step 7: Deployment

- Deploy the final dashboard using Render or Heroku
